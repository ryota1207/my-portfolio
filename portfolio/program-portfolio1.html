TIME API

■説明

  • 「TIME API」は世界都市の現在時刻を返答するAPIです。現在は世界21都市に対応しております。

    レスポンスはJSON形式で返答致します。

■構成

  • | 概要 |

    世界都市の現在時刻を返答するAPI

    | 機能 |

    世界都市の現在時刻を返答するAPI
    ├ 使用制限 : なし
    ├ 回数制限: なし
    ├ 鍵の使用 : 不要

■使用方法

  • 1.対象URLにリクエストする

    ※クエリパラメータにて利用可能な都市名を入力ください

              
          #対象URL
          http://api.web-next.cloud/time-api/v1/?city=都市名
          ※クエリパラメータにて都市名を入力ください
          ※使用できるパラメーターは下記一覧をご確認よろしくお願い致します。

          #例 東京の場合
          http://api.web-next.cloud/time-api/v1/?city=tokyo

    2.JSONデータを受け取る

              
          #返答されるJSONデータ
          { "timezone": "都市名",
            "fulldate": "yyyy-mm-dd H:i:s",
            "year": "yyyy",
            "month": "mm",
            "day": "dd",
            "hours": "H",
            "minute ": "i",
            "second": "s"
          }

          #例 東京の場合
          { "timezone": "tokyo",
            "fulldate": "2023-03-20 15:17:22",
            "year": "2023",
            "month": "03",
            "day": "20",
            "hours": "15",
            "minute ": "17",
            "second": "22"
          }

    3.プログラムに使用する
